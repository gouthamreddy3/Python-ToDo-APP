# Python To-Do List App 
 
This is a simple command-line To-Do List application written in Python. 
You can add, view, and remove tasks using this program. 
 
## How to Run 
1. Make sure Python is installed. 
2. Run the program: 
   ``` 
   python todo.py 
   ``` 
 
## Features 
- Add tasks 
- View tasks 
- Remove tasks 
- Interactive menu
